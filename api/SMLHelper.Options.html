    Component for customizing slider's value behaviour. If you need more complex behaviour than just custom value format then you can inherit this component and add it to "Slider" game object in OnGameObjectCreated event (see AddToPanel(uGUI_TabbedControlsPanel, Int32) for details on adding component) You can override value converters ConvertToDisplayValue(Single) and ConvertToSliderValue(Single), in that case internal range for slider will be changed to [0.0f : 1.0f] and you can control displayed value with these converters (also this value will be passed to OnChange<T, E>(String, T) event)
